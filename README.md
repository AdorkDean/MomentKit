# MomentKit

仿WeChat朋友圈的界面效果，因为公司项目也有类似模块，所以开发完之后，就想做个组件出来。GitHub上应该蛮多的，我也看了一些，万人烦处女座，实现效果糙，代码不整洁，就不容易看下去。自认为本组件还算整洁，希望你们能够看下去🤣🤣。MVC模式，以搭建UI和UI交互为主。如果与服务器交互的话，需要自己改一下。

### 代码结构

每个类文件中都有详细的备注和注释，模块划分和流程都还蛮清晰的，这里就不赘述了，读代码更直接一些。
